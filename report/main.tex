\begin{abstract}
\noindent
Evaluación TC3002B de Triton Heal como método \textbf{generativo-verificativo}
(verificación SLM + veto de frontera + filtrado pre-compilación).
En \Nkernels\ kernels y \Neval\ evaluaciones (6 configuraciones, Apple M4 24\,GB):
DeepSeek-Coder-V2 (frontera) MSR 87.1\%; Llama-8B local 64.5\%;
R1-Distill-14B local JSON 14.3\%; dual MSR 80.6\%, F1 0.86.
Inferencia: McNemar/Wilcoxon individuales no rechazan $H_0$ tras Holm;
Friedman global sí ($p_{\text{Holm}}<0.001$).
Speedup y generación kernel-a-kernel no medidos (limitación documentada).
\end{abstract}
