\section{Comparación crítica y robustez (2.4)}

\textbf{Comparación crítica.} En este host, el SLM local \textbf{DeepSeek-Coder-V2} (87.1\% MSR) es competitivo frente a Claude/GPT heurísticos (83.9\%). El dual maximiza sensibilidad (MSR 100\%) pero penaliza precisión global (F1 0.61) al marcar muchos \texttt{safe} como inseguros.

\textbf{Robustez.} (1) \emph{Frontera simulada:} Claude/GPT sin API. (2) \emph{Veto dual local:} no replica Claude comercial. (3) \emph{Seed único} (42). (4) \emph{Hardware:} resultados válidos para Apple M4 24\,GB, no para A100/CUDA.

\section{Amenazas observadas (2.5)}

\begin{itemize}[noitemsep]
  \item \textbf{Interna:} APIs ausentes; veto dual implementado solo con Llama local.
  \item \textbf{Externa:} benchmark de 70 kernels; un solo Mac M4.
  \item \textbf{Constructo:} MSR $\neq$ seguridad en ejecución GPU real.
\end{itemize}

\section{Discusión (2.6)}

\textbf{¿Qué evidencia hay?} Con Ollama en Metal, los SLM locales son viables (JSON 100\%, latencias de segundos). La superioridad estadística del dual sobre Llama en FNR (C3) apoya parcialmente PRI-2; PRI-1 no se confirma entre Claude heur.\ y Llama.

\textbf{Limitaciones.} Sin APIs comerciales; dual con alto MSR pero bajo F1 por conservadurismo del prompt estricto.

\textbf{¿Qué haríamos distinto?} Configurar claves API para contrastes de frontera reales; calibrar el veto para reducir falsos positivos; repetir en GPU NVIDIA para validez externa.

\section{Conclusión (2.7)}

\begin{enumerate}[noitemsep]
  \item \textbf{$H_0$ MSR (C1):} No rechazada; DeepSeek local sí supera descriptivamente a Llama y a heurísticos de frontera.
  \item \textbf{Evidencia para Triton Heal:} Dual reduce FNR vs.\ Llama (C3, Holm); trade-off en F1 por falsos positivos.
  \item \textbf{Latencia:} Local Ollama (2--8\,s) vs.\ heurístico de frontera ($\sim$80\,ms); no comparable con WAN real.
  \item \textbf{Reproducibilidad:} \texttt{make experiments stats figures pdf}; entorno M4 documentado en Tabla~\ref{tab:repro}.
  \item \textbf{Escala:} Comparación 8B vs 16B local realizada; frontera comercial pendiente de API.
\end{enumerate}
